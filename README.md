# Work in Progress

> A GitHub App built with [Probot](https://github.com/probot/probot) that makes sure if there is "WIP:" in pull request title, it will not be allowed to merge with the master branch.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
